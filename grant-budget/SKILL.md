# NRF Grant Budget Spreadsheet Template

## Overview
한국연구재단 개인기초연구(중견) 연차보고서 예산 편성용 스프레드시트 템플릿.
5개 시트가 수식으로 자동 연결됨.

## Sheet Structure

### 1. 연구원별 예산편성
- 연구원 11명 + (공용) = 12행
- 실험유형 드롭다운: 온라인/오프라인/MRI/기타
- 참여자비 소계 = 모집인원 × 1인당 참여자비 (자동)
- 인프라 비용 = SUMIF(장비시트, 연구원명, 소계열) (자동)
- 실험기간: YYMMDD 텍스트 포맷
- GPU: Y/N 드롭다운

### 2. 기존실험 지출통계
- 연구원 드롭다운 (11명 + 공용)
- 실험유형 드롭다운: 온라인/오프라인/MRI/기타
- 총 지출액 = 참여자수 × 1인당 비용 (자동)
- 프로젝트 단위로 집계

### 3. 장비구매 희망목록
- 연구원 드롭다운 (11명 + 공용)
- 분류 드롭다운: HW (데스크탑/모니터/부품), SW (라이선스/구독), AI 요금/API, 주변기기/악세사리, 실험기자재/소모품, 기타
- 제품명/모델명, 용도, 구매 링크(URL), 단가, 수량, 비고
- 소계 = 단가 × 수량 (자동)

### 4. 예산총괄
- 참여자비: 오프라인 ← SUMIF(예산편성 B="오프라인", E), 온라인 ← 지출통계 합계, MRI ← SUMIF(예산편성 B="MRI", E)
- 장비재료비: 6개 분류 각각 ← SUMIF(장비시트 B열, I열)
- 합계 = 참여자비 소계 + 장비재료비 소계

### 5. 제출일정·담당
- 마감일/항목/담당자/완료(✅⬜ 체크박스)/비고
- GRM 발표 정보: 순서/발표자/제목/Abstract/서버주소

## Cross-Sheet Connection Map
```
장비구매 희망목록 (I열 소계)
  ├→ 예산총괄 C8~C13 (6개 분류별 SUMIF on B열)
  └→ 연구원별 예산편성 F열 (12명 각각 SUMIF on A열)

기존실험 지출통계 F합계 → 예산총괄 C5
연구원별 예산편성 E합계 → 예산총괄 C4 (오프라인 SUMIF), C6 (MRI SUMIF)
```

## Visual Design
- 크림색(FFF3CD) = 필수입력
- 연회색(F0F0F0) = 자동계산/참조 (수정 불필요)
- 연민트(E8F5E9) = (공용) 행
- 폰트: Arial 11pt, 테두리: 연회색(C0C0C0)
